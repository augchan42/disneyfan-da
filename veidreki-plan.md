## Phase 1: Veidreki Token (URGENT — event ends April 3, 2026)

> **This is the time-limited piece. Cannot be obtained after the event ends.**

### How to get it

Collect **36 Stormbound Eggs from the Event Pass** → fills the Veidreki Token meter.

> **IMPORTANT:** Only Stormbound Eggs earned from Event Pass tiers count toward the token meter. Eggs from daily/repeatable missions do NOT count — those are separate (useful for Archie's Shop items).

### How Event Pass eggs work

- Event Pass resets **daily at 7:00 PM EST**
- Requires **10 missions** to complete all 10 tiers (3 dailies + 7 repeatables)
- Tiers 3, 6, 9, and 10 each give 1 Stormbound Egg = **4 Event Pass eggs/day**
- Other tiers give non-egg rewards (Growth Potion, Lucky Chest, other eggs, potions)
